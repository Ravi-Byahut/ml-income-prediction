## Project Description:
Problem statement: Created a classification model to predict whether a person makes over $50k a
year
## Context:
This data was extracted from the 1994 Census bureau database by Ronny Kohavi and Barry
Becker (Data Mining and Visualization, Silicon Graphics).
# Details of features:
The columns are described as follows:
• Age
• Workclass
• Fnlwgt
• Education
• education_num
• marital_status
• occupation
• relationship
• race
• sex
• capital_gain
• capital_loss
• hours_per_week
• native_country
• income
